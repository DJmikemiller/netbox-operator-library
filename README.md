# NetBox Operator Library

Bibliothèque NetBox Community 4.6.5 destinée aux réseaux opérateurs.

## Objectifs

- Réutiliser la NetBox Community Device Type Library
- Ajouter uniquement les équipements absents
- Maintenir une compatibilité avec NetBox 4.6.x
- Fournir des scripts d'import et de synchronisation
