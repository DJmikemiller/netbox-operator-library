# Contributing

Thank you for contributing to the NetBox Operator Library.

Our objective is to provide high-quality telecom device definitions for NetBox.

---

## General Rules

- One YAML file per Device Type.
- One YAML file per Module Type.
- Follow the official NetBox Device Type Library format.
- Use official vendor documentation whenever possible.
- Avoid duplicate definitions.

---

## Naming

Keep original manufacturer names.

Examples:

- Nokia
- Huawei
- Cisco
- Juniper Networks

Do not invent abbreviations.

---

## Documentation

Every contribution should reference the hardware documentation used to create the definition.

---

## YAML

Before submitting:

- verify YAML syntax
- verify NetBox compatibility
- verify slot numbering
- verify interface naming

---

## Pull Requests

Please describe:

- what has been added
- vendor
- model
- documentation source
- screenshots if available

---

## Issues

Use GitHub Issues for:

- missing Device Types
- missing Module Types
- corrections
- feature requests

---

Thank you for helping improve the NetBox Operator Library.
